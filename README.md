# DysLearn

## 🌟 Overview

**DysLearn** is an AI-powered learning platform designed to help children with dyslexia study more comfortably and effectively.

Students with dyslexia often face difficulties with complex text, concentration, and traditional learning environments. DysLearn addresses these challenges by providing AI-powered tools that simplify learning, improve focus, and encourage consistent study habits.

The platform integrates intelligent text simplification, AI-assisted study help, smart scheduling, gamified quizzes, and progress tracking — all designed with an accessibility-first user interface tailored for dyslexic learners.

---

## 🚀 Key Features

### ⚙️ Accessibility Settings Panel

A customizable settings panel designed specifically for dyslexic learners.

Users can adjust:

* Dyslexia-friendly fonts (OpenDyslexic, Lexend, Atkinson Hyperlegible)
* Font size
* Line spacing
* Letter spacing
* Word spacing
* Website themes
* Soothing background sounds (ocean, rain, breeze)

These adjustments allow students to create a reading environment that is comfortable for them.

---

### 🤖 AI Study Assistant

A floating AI chatbot that helps students with study-related questions.

Features:

* Clear and concise explanations
* Simplified answers for better understanding
* Quick concept clarification
* Study assistance across multiple subjects

The chatbot appears as a floating icon and expands into a chat window when clicked.

---

### 📖 Simplify Text

Students can simplify difficult study material using three input methods:

* Enter text manually
* Voice input
* Upload documents

The AI simplifies complex text by:

* Replacing difficult words with simpler alternatives
* Breaking long sentences into short ones
* Presenting content in an easier structure

Additional learning tools include:

* **Key Points** – Important ideas extracted from the text
* **Examples** – Relevant examples for better understanding
* **Mindmap** – Visual structure of the concept
* **Summary** – Short overview of the content
* **Layman Explanation** – Simplified explanation in everyday language
* **Visual Examples** – AI-generated images to explain concepts

A **Read Aloud** option allows the system to read the simplified content to the student.

---

### 📅 Smart Study Planner

An intelligent scheduling system that helps students plan their study time efficiently.

Users can:

* Add subjects
* Set study hours per day
* Assign priorities
* Upload study material

The system automatically generates a balanced study schedule with built-in break times.

Features include:

* Editable schedules
* Pomodoro timer
* Focus mode
* Completion tracking
* Rewards for completing schedules

---

### 🎮 Gamified Learning

Students can upload study material and automatically generate quizzes.

The system:

* Extracts important questions from the content
* Presents interactive quizzes
* Rewards correct answers with points

Gamification improves engagement and helps students learn through active recall.

---

### 🧘 Breaktime Activities

Breaktime features help students relax and reduce cognitive fatigue.

Activities include:

* Memory matching games
* Simon Says
* Bubble pop games
* Missing letter puzzles
* Crossword puzzles
* Breathing exercises
* Short meditation sessions
* Motivational quotes

These activities help students reset their focus before continuing their studies.

---

### 📊 Student Dashboard

A comprehensive dashboard that tracks learning progress.

Displays:

* Points earned
* Learning streaks
* Badges
* Study time
* Completed schedules
* Quiz performance
* Activity reports

Visual charts provide insights into the student's learning habits.

---

### 👨‍🏫 Teacher / Parent View

A special dashboard that allows parents or teachers to monitor student progress.

They can view:

* Study consistency
* Quiz scores
* Schedule completion
* Learning activity reports

This allows better support for students outside the platform.

---

### 💬 Encouragement System

Students can access motivational messages whenever they feel discouraged.

This feature provides:

* Encouraging quotes
* Positive reinforcement
* Study motivation reminders

## ⚙️ Getting Started

### 1️⃣ Clone the Repository

git clone https://github.com/yourusername/DysLearn.git

cd DysLearn

---

### 2️⃣ Backend Setup

cd backend

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

uvicorn main:app --reload

---

### 3️⃣ Frontend Setup

cd frontend

npm install

npm run dev

Visit the application at:

http://localhost:3000
